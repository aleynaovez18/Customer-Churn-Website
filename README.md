# 📊 TelekomAI: Müşteri Ayrılma (Churn) Analiz Portalı

TelekomAI, telekomünikasyon sektöründeki müşteri terk risklerini önceden tahmin etmek ve şirketlere stratejik kararlarında yardımcı olmak için geliştirilmiş uçtan uca bir **Makine Öğrenmesi** projesidir.

---

## ✨ Öne Çıkan Özellikler
* **🤖 Güçlü Tahminleme:** XGBoost algoritması ile optimize edilmiş yüksek başarı oranı.
* **📉 Stratejik Analiz:** %77 Recall değeri ile riskli müşterileri yüksek doğrulukla yakalama.
* **💡 Akıllı Öneri Sistemi:** Tahmin sonucuna göre müşteriye özel aksiyon planı üretimi.
* **🌐 Modern Arayüz:** FastAPI ve Bootstrap ile geliştirilmiş, kullanıcı dostu ve interaktif dashboard.

---

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Python 3.12
* **Kütüphaneler:** Scikit-learn, XGBoost, Pandas, NumPy
* **Backend:** FastAPI, Uvicorn, Pydantic
* **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript (Fetch API)
* **Paketleme:** Pickle
